# Muskan's Maths

### Premium Mathematics Learning Progressive Web Application (PWA)

**Muskan's Maths** is a modern Progressive Web Application developed by **JNR Corporations** to provide an interactive, fast, and offline-first mathematics learning experience for students. The application is built using **Pure HTML, CSS, and Vanilla JavaScript** with a strong focus on performance, accessibility, and mobile-first design.

---

## ✨ Features

* 📚 Interactive Mathematics Chapters
* 📖 Formula Vault
* 📝 Practice Questions
* 🎯 Mock Tests
* 💡 Study Tips
* 📱 Fully Responsive Mobile-First Design
* 🌐 Offline Learning Support
* ⚡ High-Performance Service Worker

## 🚀 Installation

1. Clone or download this repository.
2. Place all project files inside your web server's public directory.
3. Serve the project over **HTTPS** (recommended for full PWA functionality).
4. Open the application in a modern browser such as Google Chrome, Microsoft Edge, or Samsung Internet.
5. Install the application using the browser's **Install App** option.
